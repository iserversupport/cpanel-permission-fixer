Automated cPanel Account Permissions Fixer

This open-source Bash utility helps linux systems administrators and web hosts quickly repair misconfigured file and directory permissions that trigger 500 Internal Server Errors across cPanel environments.

How to Use the Script via SSH

wget https://raw.githubusercontent.com/iserversupport/cpanel-permission-fixer/main/fix_cpanel_perms.sh
chmod +x fix_cpanel_perms.sh
./fix_cpanel_perms.sh cpanel_username

Features

  • Safely targets specific users without risking broader system permissions.
  • Automates mass changes of directories to 755 and files to 644.
  • Verifies and enforces strict standard user ownership boundaries.
